    hexo 使用筆記

    發表於 2022-11-16 分類於 hexo

    前言：

    本篇主要用於紀錄如何使用 Hexo 框架搭配 GitHub 快速建立並架設網誌。

    介紹：

    Hexo 是個基於 node.js 的部落格框架，它能夠支援 Markdown 語法解析文章，並且透過套用的主題渲染靜態網站。

    為何使用 Hexo ：

    • 使用 markdown 語法，我可以專注於部落格撰寫，不會被樣式與語法拉開注意力。
    • 中文資源豐富，不必硬啃下其他語言所撰寫的文件。
    • 多樣化套版，可以隨心所欲切換不同的主題，且不必自行撰寫樣式。

    開始使用 Hexo ：

    以下預設已完成環境建置

    透過 npm 或 yarn 安裝 hexo-cli (我偏好使用 npm)

    $ npm install hexo-cli -g

    建立 Hexo 專案

    $ hexo init [your blog name]

    進入 你的專案資料夾

    $ cd [your blog name]

    npm 下載 hexo 所需套件

    $ npm install

    啟動 server

    $ hexo server

    完成
